# Yo! Noid: Harbor Cheese

Um jogo de plataforma side-scroller com mecânica de ioiô (grappling hook) feito em Phaser 3.

## Como Jogar
1. Abra o arquivo `index.html` em um navegador.
2. Use as **Setas** ou **WASD** para se mover.
3. Use **Espaço** para pular (Pulo Duplo disponível!).
4. Use **K ou SHIFT** perto de um ponto dourado (âncora) para prender o ioiô e balançar.
5. Cuidado com os **barcos móveis** e plataformas flutuantes!
6. Colete os pepperonis para aumentar sua pontuação!

## Tecnologias
- Phaser 3 (Matter.js Physics)
- HTML5 / CSS3 / JavaScript
